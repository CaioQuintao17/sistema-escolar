# Sistema Escolar

Sistema de gerenciamento escolar desenvolvido para estudo de desenvolvimento Full Stack.

## Tecnologias

* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* HTML
* CSS
* JavaScript

## Funcionalidades

* Cadastro de alunos
* Listagem de alunos
* Busca de aluno por ID
* Atualização de alunos
* Exclusão de alunos
* Cadastro de usuários
* Login
* Autenticação com JWT

## Estrutura do Projeto

backend/

* FastAPI
* PostgreSQL
* SQLAlchemy

front-end/

* HTML
* CSS
* JavaScript

## Como executar

1. Instale as dependências:

pip install -r requirements.txt

2. Execute a API:

uvicorn main:app --reload

3. Abra o front-end no navegador.
